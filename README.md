This project is a Library Management System implemented in Python. It allows users to manage both physical books and digital books with features such as:

Adding new books or e-books to the inventory
Searching books by title or ISBN
Issuing and returning books with status tracking
Saving and loading book records using a CSV file The system is built using Object-Oriented Programming (OOP) principles with abstract classes, inheritance, and file handling. It provides a simple menu-driven interface for easy interaction.
